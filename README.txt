# EldenRingModTranslator

《艾尔登法环》MOD翻译工具
完全无要任何编程基础，会用谷歌翻译就能10分钟内机翻汉化艾尔登法环MOD，当然也能作为
汉化大佬的辅助工具，专注于翻译，而不是折腾其他的。
(原版游戏的文本用对应的中文代替，新增加的文本使用机器翻译)
